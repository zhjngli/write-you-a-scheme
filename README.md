# write-you-a-scheme
https://en.wikibooks.org/wiki/Write_Yourself_a_Scheme_in_48_Hours

## build
1. `ghc --make -o parser parser.hs`
1. `./parser`

Can be used as a REPL or with a single filename to load and run.
